# Possible Additions

This file captures extensions that are useful but not required for v0.5 or the first v1.0.

## Detection

- brand aliases in brands CSV
- competitor alias support
- feature-level LLM judge using response text plus feature description
- confidence score per brand detection decision

## Aggregation

- engine-specific coverage and feature gaps
- date-over-date trend outputs
- confidence intervals for low sample clusters
- deduplication across repeated prompt/model runs

## Product UX

- HTML overview report
- detail page per feature cluster
- ranked opportunity score
- top competitor per feature cluster
- example cited sources in detail view

## Peec Integration

- prompt/topic filters exposed in the workflow UI
- incremental export by last successful run timestamp
- optional persistence of Peec export metadata

## Workflow / Packaging

- cron-friendly CLI wrapper
- richer n8n workflow with Slack/email summary
- MCP server resources for latest overview/detail outputs
- Docker packaging
